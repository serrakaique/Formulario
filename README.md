# Formulário
Formulário em HTML, CSS , PHP e MYSQL.

Fazendo uma área de login com cadastro e acesso, clicando no cadastro vai para a página de cadastro.html com login, senha e confirmar senha.

Tem no action.php a conexão do banco de dados para o php e o confirmar.senha.php seria tem a conexão com o action.php para quando a pessoa digitar, salvar no MYSQL e para não ter erro na senha, tem um if para comparar as duas senhas.
